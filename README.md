# React + TypeScript + Vite + Antd + Tailwind css

## Using Node:20.10.0

## How to running
- For npm
   - Install lib

   ```bash
   npm install
   ```

   - Running
   ```bash
   npm run dev
   ```
---
- For yarn
   - Install lib

   ```bash
   yarn install
   ```

   - Running
   ```bash
   yarn dev
   ```


### Lib

- Axios / antd / tailwind css

### Deploy with docker

- ```bash
  docker compose up -d --build
  ```
